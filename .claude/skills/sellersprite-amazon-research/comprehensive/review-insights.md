# 买家评论洞察

深度分析买家评论，提炼痛点和改进方向。

## 使用方式

```
/review-insights [ASIN] [站点]
```

## 工具调用

`mcp__sellersprite__review` — 扁平参数 `{"marketplace":"US", "asin":"B0XXX"}`

**响应解析**：`result.content[0].text` 是 JSON 字符串，需二次解析。

## ⚠️ review 工具限制

- **每次最多返回 20 条**（size=100 也被限制为 20）
- 评分分布是**样本**，不代表总体
- 报告中必须注明："仅展示最近 20 条评论，不代表总体分布"
- 实际字段：`content`（非 `body`）、`date`（非 `createTime`）

## 执行步骤

### 第1步: 获取评论数据

调用 `mcp__sellersprite__review` 获取 ASIN 评论数据。

### 第2步: NLP 分类分析

自行对评论内容进行 NLP 聚类：正面/负面/中性，提取痛点关键词。

### 第3步: 保存结果 + 生成报告

同时保存原始数据和报告到 `{类别目录}/`。

**报告结构：** 评论概览（注明抽样限制）→ 好评关键词 → 差评痛点频次 → 改进建议 → 竞品评论对比（如有）
