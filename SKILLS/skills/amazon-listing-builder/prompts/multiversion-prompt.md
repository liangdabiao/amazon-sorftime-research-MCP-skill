# 多版本生成提示词

> 用于第四到第六步：生成多版本标题/五点/描述供对比。

---

## 一、多版本标题提示词

```
基于以下输入，生成 3 版标题：

## 输入
- 关键词分层词库：[粘贴 P0/P1 关键词]
- 痛点-证据映射表：[粘贴 Top 5 卖点]
- 品牌名：[填]
- 产品规格：[填，如 12 bundles, 16 inches]
- 合规限制：[填，如不能用绝对化表达]
- 目标站点：[填，如 US（200 字符上限）]

## 任务：生成 3 版

### 版本 1：关键词覆盖版
特点：多埋词，结构紧凑
适用：广告冷启动、强竞价品类

### 版本 2：转化表达版
特点：可读性优先，价值前置，移动端友好
适用：自然排名沉淀、移动端流量为主

### 版本 3：简洁合规版
特点：字数克制，无夸大表达，主词前置
适用：高合规风险品类、新账号

## 每版要求
1. 遵循"品牌 + 核心词 + 差异化属性 + 主场景 + 规格"结构
2. 字符 150-180（美国站）
3. 前 80 字符含核心词 + 主卖点
4. 标注埋入的关键词清单（对照词库）
5. 列出优缺点 + 适用场景

## 输出

### 版本 1：关键词覆盖版
标题：[文本]
字符数：[X]
前 80 字符：[文本]
埋入关键词：[L1/L2/L3/L5 列表]
优点：[...]
缺点：[...]
适用：[...]

### 版本 2：转化表达版
...

### 版本 3：简洁合规版
...

### 推荐基础版
选择：[版本 X]
理由：[根据品类、账号阶段、广告策略选择]
```

## 二、多版本五点提示词

```
基于以下输入，生成 2 版五点描述：

## 输入
- 痛点-证据映射表：[粘贴]
- 关键词分层词库：[粘贴]
- 用户问题库 Top 5：[粘贴]
- 已选标题版本：[粘贴]
- 合规限制：[填]

## 任务：生成 2 版

### 版本 A：转化决策链版（推荐）
每点严格对应一个决策环节：
- Point 1：核心价值（为什么适合主场景）
- Point 2：痛点解决（最大担忧）
- Point 3：使用场景（具体）
- Point 4：规格使用
- Point 5：信任风险（包装/维护）

### 版本 B：场景叙事版
每点对应一个主场景：
- Point 1：patio 场景下的体验
- Point 2：porch 场景下的体验
- Point 3：garden 场景下的体验
- Point 4：planter 使用方式
- Point 5：跨场景共性卖点

## 每点要求
1. 加粗卖点标题（5-8 词）
2. 1 句承诺 + 1-2 句证据
3. 埋入 3+ 关键词（来自不同层级）
4. 字符 200-300
5. 对应一个用户问题
6. 禁用空泛词：premium / perfect / easy / great gift / 100% / lifetime

## 输出
[两版完整五点对比，每点标注埋入关键词、对应问题、合规检查]

## 对比分析
- 哪版更适合主推？
- 两版各自的优缺点？
- 推荐基础版及理由？
```

## 三、多版本描述/A+ 提示词

```
基于以下输入，生成 2 版描述 + A+ 结构：

## 输入
- 痛点-证据映射表：[粘贴]
- 五点描述（已定稿）：[粘贴]
- 品牌定位：[填]
- 产品规格：[填]

## 任务

### 版本 A：故事叙事版（推荐低维护品类）
按 7 段式：
1. 开头痛点共鸣
2. 使用场景代入
3. 产品解决方案
4. 细节对比
5. 使用步骤
6. 适合人群
7. 注意事项

### 版本 B：模块化对比版（推荐技术参数强的品类）
按 6 模块：
1. 核心卖点（3 句话总结）
2. 应用场景（4-6 个场景）
3. 规格表（表格化）
4. 使用步骤
5. 对比表（vs 普通款）
6. 注意事项

## A+ 模块（两版共用）
- 第 1 屏：户外场景大图
- 第 2 屏：核心材料（如抗 UV）
- 第 3 屏：细节特写
- 第 4 屏：场景拼图
- 第 5 屏：规格对比
- 第 6 屏：包装加固
- 第 7 屏：普通款 vs 升级款对比

## 输出
- 描述版本 A + 版本 B
- A+ 7 屏结构（每屏：标题、正文、图片需求、关键词）
- 推荐版本及理由
```

---

## 使用建议

1. **多版本对比是核心** — 不要只生成一版就上线
2. **至少 3 版标题** — 因为标题对系统理解和点击率影响最大
3. **五点至少 2 版** — 一版按决策链、一版按场景叙事
4. **描述 2 版** — 故事版适合情感品类，模块版适合技术品类
5. **由人最终决策** — AI 列优缺点，人选基础版
